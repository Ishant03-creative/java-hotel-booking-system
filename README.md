# Hotel Booking Management System

A Java console-based Hotel Booking Management System that allows users to manage room reservations efficiently through a menu-driven interface.

## 🚀 Features

* View available and booked rooms
* Book rooms with customer name
* Cancel existing bookings
* Search booking by customer name
* Room categories (Standard, Deluxe, Suite)
* Dynamic room pricing
* Booking receipt generation
* Input validation for room numbers and menu choices

## 🛠️ Technologies Used

* Java
* IntelliJ IDEA
* Git & GitHub

## ▶️ How to Run

1. Open the project in IntelliJ IDEA
2. Navigate to `src/Main.java`
3. Run the program
4. Use menu options to interact with the system

## 📋 Menu Options

1. View Rooms
2. Book Room
3. Cancel Booking
4. Search Booking
5. Exit
